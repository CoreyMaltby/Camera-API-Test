[```MediaDevices.getUserMedia()```](https://developer.mozilla.org/en-US/docs/Web/API/MediaDevices/getUserMedia): Prompts the user for permission to use a media input. Returns a ```MediaStream``` object which can be put in a HTML video tag.

[```HTMLMediaElement.srcObject```](https://developer.mozilla.org/en-US/docs/Web/API/HTMLMediaElement/srcObject): Used to set or recieve the video stream into the ```<video>``` element in real-time.

[Canvas Rendering](https://developer.mozilla.org/en-US/docs/Web/API/CanvasRenderingContext2D): Used to draw the video frame onto an invisible digital canvas so indivudal pixels can be looked at.
